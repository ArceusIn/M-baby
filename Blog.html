Sending folders online with ToffeeShare

It will most likely be very slow Anyone who's ever copied some files from a computer to a flash drive will know that copying a lot of small files will generally take more time than to send one large one

Multiple transfers at once and getting ready for mobile

In our last update we asked you what features you would like to see next. There wasn't really a clear winner, so we went ahead and chose for you :)The first new update makes it possible to transfer to multiple people at the same time without them having to wait. You can see if there's multiple active transfers when dots appear underneath the progress bar. You can then scroll through them to see the actual progress. This is a temporary solution, since we're working on a whole new design to get a clearer overview.

When you share a file, we use our new shorter domain toffeesha.re. This makes for a cleaner url, but more importantly allows us to stay in the clear when Chrome scans your files (yes Chrome does that) and marks it as undesirable content on our website. This had a negative impact on our search score, so hopefully that penalty will only affect this new domain.

First steps have been set to restart failed transfers from a certain point, so that you don't have to redownload the entire file. This should make sharing large files less cumbersome on unstable internet connections. For a fully working implementation youâ€™ll have to wait a bit longer.
